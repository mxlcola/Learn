# **① 冲击成本型（大资金建仓/调仓导致的波动）**

> 本质：大资金 **必须买** → 推高价格；必须卖 → 打低价格。

### ♟ 机构怎么赚钱

* 公募/专户必须按指令建仓
* 建仓时造成 **急拉 → 回落 → 尾盘再扫** 的波动
* 真正吃这段波动的是
  **[[老鼠仓]] / 关联账户 / 内部生态资金**

机构本身不靠这赚钱，它只是“完成任务”。

### 📟 盘面信号

* 集合竞价直接顶上 1–3%
* 9:30–9:40 **直线猛拉到 +5%/+6%**
* 之后回落贴近黄线
* 尾盘 14:30 **突然大单急扫**

### 🧩 散户关系：极弱（别碰）

你不是它对手盘，它也不是来吃你。

### 🎯 散户应对

* 遇到**突然直线脉冲**：
  → **摸一把就跑**
* 别幻想跟着它吃波段
* 你看到的都是“别人建仓造成的噪音”
