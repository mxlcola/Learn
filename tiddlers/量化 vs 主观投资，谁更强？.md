# 📘 **量化 vs 主观投资，谁更强？**

### 取决于市场风格：

| 市场环境                               | 谁表现更好   | 原因                        |
| -------------------------------------- | ------------ | --------------------------- |
| **结构性牛市**（如2020年核心资产大牛） | **主观多头** | 可押重赛道获取巨大利润      |
| **震荡市或熊市**                       | **量化策略** | 分散化 + 高频低买高卖更优势 |

**近3年数据：**
 量化整体胜过主观，尤其“赢麻了”。但这不是必然，只是市场风格的结果。
